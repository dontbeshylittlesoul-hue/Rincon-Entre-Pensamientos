# Rincón entre Pensamientos

A static, immersive digital reading experience built in Spanish. The site evokes a quiet, candlelit corner for slow, contemplative reading.

## Technologies

- Plain HTML5, CSS3, JavaScript (no frameworks or build tools)
- Google Fonts: Cormorant Garamond (display) + Crimson Pro (body)
- Netlify static hosting

## Running locally

Open `index.html` directly in a browser, or serve it with any static file server:

```bash
npx serve .
```

## Audio

Replace `ambiente.mp3` in the project root with your own rain / fire / ambient audio file. The comment in `index.html` marks the exact `<source>` tag to update.
